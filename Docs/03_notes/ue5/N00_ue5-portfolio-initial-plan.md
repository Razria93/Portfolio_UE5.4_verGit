# 포트폴리오 구성 순서
---

### 1단계: 기본 시스템 구축

1. Debugging Level 구성

2. 플레이어 구성 (구현순서 1)
	- 매시 / 카메라
	- 레벨 배치
	- 기본 이동 + 카메라 조작
	- idle / walk / sprint / jump
   
3. 플레이어 기본 전투 구성 (구현순서 2)
	- 무기 장착/해제
	- 공격 애니메이션
	- 충돌체

---

### 2단계: 기본 전투 루프 구현	   

1. 적 기본 구성 (구현순서 3)
	- 매시 / 애니메이션 / 충돌체

2. 기본 UI 구성 (구현순서 4)
	1. 플레이어/적 HP 및 상태

3. 전투 시스템 기본 구성  (구현순서 5)
	1. Hit 및 HitReact
	2. HP 시스템
	3. 데미지 전달 시스템
	4. 적 Death + Destroy

4. 플레이어 Hit 및 Death 구성 (구현순서 6)
	1. 플레이어 HIt 및 React
	2. 플레이어 Death + Destroy

5. 타겟팅 시스템 구성  (구현순서 7)
	1. 타겟팅 시스템 구성

---

### 3단계: AI 구성

1. 팀 구성  (구현순서 8)
	1. 플레이어 / 중립 / 적

2. 적 AI 구성 - 이동  (구현순서 9)
	1. 지점 이동 / 랜덤 쿼리 이동 / 순환 이동 / 추적 이동 구성
		- AI / 네비게이션 / 쿼리 이용
	
3. 적 AI 구성  - 경계  (구현순서 10)
	1. 추적 후 분산 알고리즘
	2. 전투 우선순위 부여 (공격대기 / 경계대기 / 수비대기)
	3. 플레이어 주시

4. 적 AI 구성 - 공격  (구현순서 11)
	1. 공격 애니메이션 부여
	2. 데미지 전달 시스템 적용
	3. 콤보 시스템 및 애니메이션 부여
	4. 거리별 공격 분기 및 랜덤값에 따른 알고리즘

5. 플레이어 가드 / 패링 / 회피 시스템 구성  (구현순서 12)
	1. 플레이어 가드 시스템 구성
		- 가드 / 퍼펙트 가드 / 가드 브레이크
	2. 플레이어 패링 시스템 구성
		- 패링 / 퍼펙트 패링
	3. 플레이어 회피 시스템 구성
		- 회피 / 퍼펙트 회피

6. 적 AI 가드 / 패링 / 회피 시스템 구성  (구현순서 13)
	1. 적 가드 시스템 구성
		- 가드 / 가드 브레이크
	2. 적 패링 시스템 구성
		- 패링
	3. 적 회피 시스템 구성
		- 회피

---

### 4단계: AI 시스템 확장

1. 아군 AI 구성  (구현순서 14)
	1. 적 AI에 구성해둔 기능 그대로 차용

2. 적군 AI / 아군 AI 집단전 구성 (구현순서 15)
	1. 피아 식별
	2. AI간 전투 매칭 시스템
	3. AI 킬 이후 추가 행동 시스템

### 5단계: 아이템 / NPC / 상호작용 구성

1. 아이템 상호작용 및 사용 시스템 (구현순서 16)
2. 인벤토리 시스템 (구현순서 17)
3. 마을 구현 (구현순서 18)

5. 상점 시스템 (구현순서 20)
6. 퀘스트 시스템 (구현순서 21)

8. 레벨 이동 (구현순서 23)

---

### 6단계 : 애니메이션 고도화

1. 풋 IK / 핸드 IK (구현순서 24)

2. ALS 시스템 (구현순서 25)

3. 파쿠르 / 환경 상호작용 시스템 (구현순서 26)

4. 물리기반 애니메이션 (구현순서 27)
	- 피격 리액션
	- 던지기
	- 밀기
	- 래그돌 블랜딩 (부위별 리깅 포함)

5. 지형 / 환경감지 시스템 (구현순서 28)
	- Ground Detection (풋 IK와 연동)
	- Climbable Surface Detection (파쿠르와 연동)
	- 물리 / 환경 기반 상호작용을 위한 감지모듈

6. 특수 연출 시스템 (구현순서 29)
	- 슬로우 모션 / 타임 디스토션 (전투나 파쿠르 연출 강화)
	- 카메라 연출 (카메라 흔들림, 줌, 보정(예: 타겟 락온))

### 7단계 : VFX 구현
###### **전투 연출 VFX**
1. 히트 이펙트 / 크리티컬 이펙트 - 공격/피격 시 파티클 연출 (난이도: 낮음 / 구현순서 30)
2. 데미지 넘버 FX - 피격 숫자와 간단한 시각 효과 (난이도: 낮음 / 구현순서 31)
3. 차징 이펙트 – 스킬 모으기 시 에너지 집중 효과 (난이도: 낮음 / 구현순서 32)
4. 상태이상 VFX – 화상, 빙결, 감전 등 캐릭터 상태 표현 (난이도: 중간 / 구현순서 33)
5. 궁극기 연출 – 컷신, 카메라 흔들림, 대규모 파티클 (난이도: 높음 / 구현순서 39)
###### **환경 반응형 VFX**
1. 발자국 / 먼지 / 물 튀김 – 지형 타입별 상호작용 파티클 (난이도: 중간 / 구현순서 34)
2. 파괴 VFX – 오브젝트 파괴 시 파편, 먼지, 연기 (난이도: 높음 / 구현순서 40)

###### **카메라 & 화면 효과 VFX**
1. 슬로우모션 & 충격파 – 시간 왜곡 + 화면 Post Process (난이도: 중간 / 구현순서 35)
2. 림라이팅 / 실드 효과 – Outline, 보호막 반짝임 (난이도: 중간 / 구현순서 36)
###### **UI & HUD 연동 VFX**
1. 레벨업 / 경험치 연출 – HUD와 캐릭터 파티클 결합 (난이도: 중간 / 구현순서 37)
2. 데칼 효과 – 총알 자국, 발자국, 피 튀김 표현 (난이도: 낮음 / 구현순서 38)

###### **특수 효과 VFX**
1. 중력 / 블랙홀 연출 – 벡터장을 통한 흡입 효과 (난이도: 높음 / 구현순서 41)
2. 네트워크 싱크 VFX – 멀티플레이 환경에서 이펙트 동기화 (난이도: 매우높음 / 구현순서 42)

### 10단계 : 전투 시스템 고도화

###### 최소 구현 (필수)
1. 크리티컬 시스템 – 치명타 판정
2. 상태이상 시스템 – 화상, 빙결, 감전 등
3. 스테이터스 시스템 – HP/MP/스태미너 관리
4. 경험치 및 레벨업 시스템 – 성장 로직

###### 확장 구현 (선택)
1. 차징어택 - 차징 공격 (위에 차징 이펙트랑 함께 구현)
2. 백어택 – 뒤 공격 보너스
3. 그로기 – 경직/다운 처리
4. 정밀한 히트 리액션 – 부위별 피격 반응
5. 공중 연속 공격 – 공중 콤보

---

### 11단계 : 스킬 & 무기 시스템

###### 최소 구현 (필수)
1. 무기 에셋 관리 – 무기 데이터/리소스 관리
2. 스킬 에셋 관리 – 스킬 데이터/리소스 관리
3. 궁극기 – 전투의 핵심 연출

###### 확장 구현 (선택)
1. 아이템 특수작용 – 특수 아이템 효과
2. 추적 – 호밍/자동 타겟팅 스킬
3. 소환 – 소환수/오브젝트 생성

---

### 12단계 : 이동 & 파쿠르

###### 최소 구현 (필수)
1. 파쿠르 / 낙법 – 벽타기, 장애물 넘기, 착지 충격 완화

###### 확장 구현 (선택)
1. 은신 – 은폐 및 탐지 회피
2. 비행 – 공중 이동 모드

---

### 13단계 : 환경 & 월드

2. 맵 – 작은 샘플 맵/레벨 디자인

###### 확장 구현 (선택)
1. 나나이트 – 렌더링 최적화 데모 수준

### 14단계 : 연출 & UI

###### 최소 구현 (필수)
1. 위젯 – HUD, 메뉴 구성
2. 디버깅 UI – 상태 확인용 인터페이스
3. 카메라 충돌 – 카메라와 오브젝트 겹침 방지

2. 벽 투시 – 캐릭터 가림 방지 처리
3. 에디터 구성 – 툴/플러그인 확장
